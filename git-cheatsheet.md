# Git Cheat Sheet

## Git: Configurations

###Set User Name  

```
$ git config --global user.name "FirstName LastName"
```
###Set User Email  

```
$ git config --global user.email "your-email@email-provider.com"
```
###Turn on UI Colors  

```
$ git config --global color.ui true
```
###Show current git settings  

```
$ git config --list
```
